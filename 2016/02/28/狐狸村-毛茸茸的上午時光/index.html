狐狸村 | 毛茸茸的上午時光

狐狸村光聽名字就覺得一定是超可愛的地方，本來興高采烈把它加入行程，卻發現要去狐狸村只有巴士和開車兩個選擇，然後，我們要去的時間，巴士停駛…(((ﾟДﾟ;)))


距離 3 月不久前，左思右想煩惱著到底要不要去，考量貝貝從沒在日本開過車 + 不熟悉的右駕 + 下雪的山路，感覺就很不妙，這樣真的要去嗎?(崩潰)


最後我們還是..出發了(´艸`) (貝貝看我太煩惱，決定勇敢嘗試第一次的日本自駕)

1/1000 f/3.5 ISO200

1/1250 f/3.5 ISO100

進去參觀狐狸有幾個要點要注意：

  1. 不能摸狐狸，會被咬
  2. 不能帶繩子類的東西，會誤以為是食物(所以我很小心的抓著 Gopro 下面的繩子)
  3. 避免磨擦塑膠袋的聲音，會吸引他
  4. 不可以從包包拿東西，會誤以為是食物
  5. 不可以蹲下(重要！很大機率會被咬喔)

雖然實際走過，園區內的狐狸沒有這麼可怕(看上面好像很容易被咬)，但還是要遵守園區的規定，避免發生意外狀況
